📊 Data Management

🍼 How to Use BabyLog

📱 Quick Start

Tap the emoji buttons at the bottom to record baby's activities instantly. The app automatically saves the time and date for you!

🍼 Feed - Record feeding times
💩 Poo - Track bowel movements
🧷 Nappy Change - Log diaper changes
💊 Vitamins - Track daily vitamins (once per day)

📅 Viewing Your Data

  • Today's Activities: See all of today's events at the top
  • Weekly Calendar: View activities across the week
  • Navigate Weeks: Use ← Previous / Next → buttons
  • Vitamin Status: Shows if vitamins were given today

✏️ Editing & Deleting

  • Edit Time: Click on any time to adjust it
  • Delete Events: Click the × button next to any event
  • Time Picker: Use the popup to set exact date and time

💾 Backup & Restore

Important: Your data is saved in your browser, but it's smart to backup regularly!

  • 📥 Download CSV: Save all your data to a file
  • 📤 Upload CSV: Restore from a backup file
  • Replace vs Merge: Choose to replace all data or add to existing
  • 🗑️ Clear All: Delete everything (use with caution!)

💡 Pro Tips

  • Export your data weekly as a backup
  • The app works offline once loaded
  • Add to your phone's home screen for quick access
  • Each device needs its own data - use CSV to transfer
